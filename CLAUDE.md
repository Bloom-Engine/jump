# CLAUDE.md

## Project Overview

Bloom Jump is a classic 2D side-scrolling platformer built with Bloom Engine, compiled to native by the Perry TypeScript compiler. Features pixel art sprite sheets, multiple levels, and a standalone level editor.

**Tech stack:** TypeScript (Perry-compiled) + Bloom Engine (Rust native layer with wgpu rendering)

## Repository Structure

```
jump/
  src/main.ts        # Game (single-file, Perry-safe)
  src/editor.ts      # Standalone level editor (single-file, desktop-only)
  assets/sprites/    # PNG sprite sheets (generated by tools/generate-assets.js)
  assets/sounds/     # WAV sound effects (generated by tools/generate-assets.js)
  levels/            # Level data files (.txt format)
  tools/             # Build tools (asset generator)
  android/           # Android Gradle project (BloomActivity + JNI bridge)
  build-android.sh   # Android build script (compile + APK + install)
  perry.toml         # Perry build config
```

## Build Commands

```bash
# Generate assets (once, requires Node.js)
node tools/generate-assets.js

# Build and run game (macOS)
perry compile src/main.ts -o jump && ./jump

# Build and run editor (macOS, desktop-only)
perry compile src/editor.ts -o editor && ./editor

# iOS Simulator
perry compile --target ios-simulator --features ios-game-loop src/main.ts -o bloom_jump
/usr/libexec/PlistBuddy -c "Delete :UIApplicationSceneManifest" bloom_jump.app/Info.plist
xcrun simctl install booted bloom_jump.app
xcrun simctl launch booted com.bloom.jump

# iOS Device
perry compile --target ios --features ios-game-loop src/main.ts -o BloomJump
/usr/libexec/PlistBuddy -c "Delete :UIApplicationSceneManifest" BloomJump.app/Info.plist
codesign --force --sign "Apple Development: ..." --entitlements ent.plist BloomJump.app

# tvOS
perry compile --target tvos --features ios-game-loop src/main.ts -o bloom_jump_tv

# Android (build + install + run)
./build-android.sh --run
```

## Perry Compiler Pitfalls

- **`Math.max`, `Math.min`, `Math.pow` return boxed objects** — use inline: `let r = a; if (b > r) r = b;`
- **Integer literal `0` infers i32** — always use `0.0` for float arithmetic
- **Module-level `let` mutation may not propagate** — use array-based state: `const P = [0.0, 0.0, ...]`
- **`for (const x of array)` may not work** — use index-based: `for (let i = 0; i < arr.length; i = i + 1)`
- **String `===` in switch/case may fail** — use numeric constants with if/else
- **`export const alias = fn` causes duplicate symbols** — use wrapper functions

## Level File Format

Simple text format stored in `levels/`:
```
name:Level Name
width:100
height:15
bg:0
spawn:3,12
tiles:0,0,0,1,1,1,...
entities:1,20,12;10,30,12;20,95,12
```

Tile types: 0=air, 1=grass, 2=dirt, 3=brick, 4=stone, 5=spike-up, 6=spike-down, 7=platform
Entity types: 1=walker, 2=flyer, 3=chaser, 10=coin, 11=gem, 12=spring, 20=flag

## Architecture Notes

- Both game and editor are single-file Perry apps (avoids cross-module issues)
- All mutable state uses `const` arrays for Perry safety
- Sprite sheets use 16x16 base tiles rendered at 2x (32x32 display) via drawTexturePro
- setTextureFilter(tex, FILTER_NEAREST) for crisp pixel art
- Physics: separated-axis AABB collision, 60 FPS fixed timestep

## Cross-Platform Input

The game supports three input methods:
- **Desktop**: Keyboard (Arrow Keys / WASD + Space)
- **iOS/Android**: Touch controls (virtual joystick left, jump button right, pause top-right)
- **tvOS**: Gamepad / Siri Remote (d-pad + button A for jump, Menu for pause)

Platform detection via `isMobile()` and `isTV()` from bloom/core. Touch state stored in `TCH[]` array, gamepad state in `GP[]` array. Screen size queried each frame via `getScreenWidth()`/`getScreenHeight()`.

## iOS / Android Notes

- iOS asset paths are auto-resolved to app bundle by native layer's `resolve_path()`
- Android assets extracted from APK to filesDir; `BLOOM_ASSET_PATH` env var set before loadLibrary
- `UIApplicationSceneManifest` must be deleted from Info.plist after iOS builds
- iOS provisioning profile needed at `/Users/amlug/.perry/com_bloom_jump.mobileprovision`
